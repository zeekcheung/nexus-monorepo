# Curing Monitor

This is a Cloudflare Page that displays curing data.

## Usage

The API can be accessed via below URL:

- [https://curing.zeekcheung.workers.dev/monitor](https://curing.zeekcheung.workers.dev/monitor)
- [https://curing.zeekcheung.ip-ddns.com/monitor](https://curing.zeekcheung.ip-ddns.com/monitor)

```sh
https://curing.zeekcheung.ip-ddns.com/monitor?curingType=喷雾养护&boxNumber=①&statusLabel=状态：&statusValue=空闲&temperatureLabel=温度：&temperatureValue=25&temperatureUnit=℃&humidityLabel=湿度：&humidityValue=96&humidityUnit=%&processLabel=时长：&processHourValue=5&processHourUnit=小时&processMinuteValue=30&processMinuteUnit=分钟&processSecondValue=59&processSecondUnit=秒&processSecondHidden=0&totalHourValue=12&totalMinuteValue=0&totalSecondValue=0&progressLabel=进度：&progressUnit=%&updateInterval=1
```
